* hugo-modus

An opinionated Hugo theme using the [[https://protesilaos.com/emacs/modus-themes-colors][colour palette of the Modus themes]].

** Change log

- [X] add support of MathJax
- [X] better table support

** Installation

1. Install the theme in your project

#+BEGIN_SRC shell
git submodule add https://github.com/zkazure/hugo-modus.git themes/hugo-modus
#+END_SRC

2. Configure the theme in =hugo.toml=

#+BEGIN_SRC
theme = "hugo-modus"
#+END_SRC

** Usage

*** New section

#+BEGIN_SRC shell
hugo new -k section posts/_index.md
#+END_SRC

*** New content

#+BEGIN_SRC shell
hugo new posts/your-first-post.md
#+END_SRC

** Development

The theme is styled with Tailwind CSS 4. The site loads the compiled
=assets/css/dist/main.css=, so after editing the source
=assets/css/main.css= you must rebuild it.

| Command      | Description                                                |
|--------------+------------------------------------------------------------|
| =make dev=   | Start the Hugo server against =exampleSite= for previewing |
| =make watch= | Watch =main.css= and rebuild on every change               |
| =make build= | Compile =main.css= once, optimized for production          |
